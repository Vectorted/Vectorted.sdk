## NodeRuntime API

### Global-Java-Native Objects

#### vectorted
`static read-only vectorted: VectortedModule`

A Mapping to Java class object VectortedModule,+exposing underlying IEC61850/IEC60870-104 interface functions of JNI interface.

Type: <a href="">VectortedModule</a>

#### v8Runtime
`static read-only v8Runtime: V8Block`

Used to operate V8 underlying objects and expose some wrapper layer methods.

Type: <a href="">V8Block</a>

#### $java
`static read-only $java: V8Java`

Expose certain wrapper layer methods to the V8Runtime environment.

Type: <a href="">V8Java</a>
